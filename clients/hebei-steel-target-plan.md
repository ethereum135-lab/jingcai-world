# 首单目标客户方案 - 河钢集团

## 客户基本信息

| 项目 | 内容 |
|------|------|
| 客户名称 | 河钢集团有限公司 |
| 客户类型 | 大型国有钢铁企业 |
| 总部地址 | 河北省石家庄市 |
| 年产能 | 约4500万吨钢 |
| 年铁矿石采购量 | 约6000万吨（进口约4000万吨） |
| 主要港口 | 曹妃甸港、黄骅港、天津港 |
| 首单建议 | 10万吨铁矿石 |

---

## 首单方案建议

### 商品规格
- **品种**：PB粉（Pilbara Blend Fines）或纽曼粉
- **数量**：10万吨（±10%）
- **品位**：Fe 61% 以上
- **交货方式**：CIF 曹妃甸港

### 商务条款建议
- **价格基准**：普氏62%铁矿石指数（Platts IODEX）
- **定价方式**：点价或固定价（根据客户偏好）
- **付款方式**：信用证（L/C at sight）或TT预付
- **交货期**：合同签订后30-45天

### 竞争优势
1. **国企背书**：成都国贸集团，300亿资金实力
2. **灵活付款**：可根据客户需求提供账期支持
3. **稳定供应**：与澳洲主流矿山直接合作
4. **专业服务**：全流程跟踪，质量保障

## 所需支持

1. **引荐资源**：需要两位矿业专家或成都国贸提供河钢采购负责人引荐
2. **公司材料**：成都国贸正式公司介绍、资质证明
3. **价格支持**：首单可适度让利，建立合作关系
4. **法务支持**：合同条款审核
