# 捕海翁电商运营 Skill

## 核心功能

1. **竞品监控** - 追踪对标账号/店铺数据
2. **数据分析** - 销售数据、流量数据、转化漏斗
3. **投放优化** - 千川/随心推/多多推广优化
4. **促销策划** - 活动方案、优惠券策略、满减设计
5. **多平台管理** - 抖音/拼多多/视频号协同运营

---

## 竞品监控维度

### 抖音竞品追踪
- 账号粉丝增长趋势
- 视频播放量/互动率
- 直播场次/GMV/场观
- 爆款商品/价格带
- 投放素材/文案

### 拼多多竞品追踪
- 店铺销量/评价
- 爆款SKU/价格
- 活动参与情况
- 推广投入估算

---

## 数据分析框架

### 日常数据看板
```
【流量指标】
- 曝光量、点击率、转化率

【销售指标】
- GMV、订单量、客单价、毛利率

【投放指标】
- 消耗、ROI、CPM、CPC、CPA

【内容指标】
- 播放量、互动率、涨粉数、完播率
```

### 周度复盘模板
1. 本周数据概览（环比/同比）
2. 爆款分析（成功因素提炼）
3. 问题诊断（流量/转化/客单）
4. 下周优化方向

---

## 投放优化策略

### 千川投放4阶段
1. **人气计划** - 拉新、建立模型
2. **精准转化** - 收割意向人群
3. **高客单收割** - 提升客单价
4. **放量冲刺** - 最大化GMV

### 多多推广策略
- 搜索推广：精准关键词、高出价抢位
- 场景推广：低价引流、拉升权重
- 放心推：全店托管、稳定ROI

- 数据分析必须给出：问题诊断、优化建议、预期效果
