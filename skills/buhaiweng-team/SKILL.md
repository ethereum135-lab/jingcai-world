# 捕海翁团队管理 Skill

## 核心功能

1. **日报汇总** - 自动汇总4个Agent每日报告
2. **周会整理** - 生成周会纪要和下周计划
3. **任务追踪** - 管理待办事项和完成进度
4. **决策记录** - 记录关键决策和待决策事项
5. **进度看板** - 可视化展示项目整体进度

---

## 团队架构

| 代号 | 职责 | 汇报时间 |
|------|------|---------|
| 内容官 | IP定位、内容策略、脚本产出 | 每日22:00 |
| 运营官 | 多平台运营、数据分析、投放 | 每日22:00 |
| 直播官 | 直播间搭建、话术、投流 | 每日22:00 |
| 产品官 | 供应链、选品、成本控制 | 每日22:00 |

## 自动化规则

1. **每日22:30** - 自动汇总当日日报，发送给老板
2. **每周日22:00** - 自动整理周会纪要
3. **关键决策** - 实时记录，标注待决策/已决策
4. **逾期提醒** - 任务逾期自动预警
