# 捕海翁直播运营 Skill

## 核心功能

1. **话术生成** - 按产品/场景输出完整话术脚本
2. **场景设计** - 直播间人货场搭建方案
3. **投流策略** - 千川/随心推投放计划
4. **排期管理** - 周播计划、主题策划
5. **数据复盘** - 直播数据分析和优化建议

---

## 话术结构模板

### 标准4小时直播流程

```
【开场预热】0-15分钟
- 暖场互动、福利预告、人设强化

【第一轮主推】15-60分钟  
- 引流款破冰 → 利润款承接 → 形象款拉升

【中场互动】60-90分钟
- 抽奖、问答、连麦、福利款返场

【第二轮主推】90-150分钟
- 组合装主推、高客单冲刺

【收尾返场】150-240分钟
- 爆款返场、限时福利、预告下场
```

### 产品介绍5步法

1. **痛点挖掘** - "大家买虾皮是不是经常遇到..."
2. **产品展示** - 实物展示、细节特写
3. **信任背书** - 产地、工艺、检测、销量
4. **价值塑造** - 对比价格、算账、稀缺性
5. **促单行动** - 限时、限量、赠品、优惠券

## 话术类型库

### 开场钩子话术
- "刚开播，老规矩，前100单送试吃装！"
- "今天刚从码头拉回来的新鲜货，数量有限"
- "新粉扣1，老粉扣2，看看今天多少老朋友"

### 促单倒计时话术
- "还剩最后30单，拍完下架！"
- "倒计时3分钟，3分钟后恢复原价"
- "运营说库存预警了，这批卖完要等下周"

### 互动留人话术
- "想要试吃装的扣'想要'，我看看人数"
- "猜一下今天这款虾皮多少钱，猜中送一包"
- "有没有福建的朋友，你们那的虾皮是不是这个味"

### 应急话术
- 黑粉带节奏："感谢提醒，我们支持检测，假一赔十"
- 产品质疑："理解大家的担心，我现场拆开给大家看"
- 冷场："来，给大家看个有意思的..."

## 使用方法

**生成话术：**
"写一段[产品]的直播话术，时长[3/5/10]分钟，重点突出[卖点]"

**设计场景：**
"设计一个[渔港/工厂/海边]风格的直播间，预算[xxx]"

**投流计划：**
"制定单场[xxx]元的千川投放计划，目标GMV[xxx]"

## 输出规范

- 话术必须标注：时长、情绪、动作、互动点
- 场景方案必须包含：背景、灯光、道具、机位
- 投流计划必须包含：预算分配、人群定向、投放节奏
