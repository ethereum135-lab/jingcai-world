# 捕海翁内容策划 Skill

## 核心功能

1. **人设定位** - 根据品牌调性输出IP人设方案
2. **爆款选题** - 生成热点选题、系列选题、日常选题
3. **脚本生成** - 按模板输出完整视频脚本（钩子+内容+转化）
4. **竞品分析** - 拆解对标账号爆款逻辑
5. **排期管理** - 制定周/月内容日历

## 爆款选题公式

### 选题类型
1. **人设故事类** - 创业经历、渔民生活、选品故事
2. **避坑指南类** - 如何挑选、如何保存、如何烹饪
3. **工厂探秘类** - 捕捞现场、加工过程、质检环节
4. **产品测评类** - 开箱试吃、对比评测、吃法创新
5. **海边日常类** - 赶海vlog、渔港生活、海鲜美食

### 爆款标题公式
- 数字法：「3招教你挑到最鲜的鱿鱼丝」
- 对比法：「超市30块的 VS 我们9块9的」
- 悬念法：「渔民从来不吃这种海鲜，原因竟是...」
- 痛点法：「你买的虾皮可能全是添加剂」
- 反常识：「越贵的海鲜越不新鲜？」

## 脚本结构模板

```
【视频时长】：30-60秒
【开场钩子】（3秒）：抓眼球，留人
【内容主体】（20-40秒）：干货/故事/展示
【互动引导】（3秒）：评论/点赞/关注
【转化引导】（3秒）：小黄车/橱窗/私信
```

## 使用方法

**生成脚本：**
"给我写一条关于[产品]的[类型]脚本，风格[人设方案]"

**策划选题：**
"本周选题规划，目标[涨粉/带货/互动]，平台[抖音/视频号]"

**竞品分析：**
"分析抖音账号[@xxx]，拆解其爆款逻辑"

## 输出规范

- 脚本必须包含：标题、时长、场景、台词、镜头、BGM建议
- 选题必须标注：预估热度、制作难度、转化潜力
- 所有输出使用中文，符合抖音/视频号平台调性
