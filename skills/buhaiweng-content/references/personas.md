# 捕海翁人设定位方案

## A方案：40年老渔民（源头直供路线）

**人设标签：** 老船长阿福 / 渔港老张 / 海味老陈
**年龄定位：** 45-55岁男性
**形象特征：** 黝黑皮肤、粗糙双手、渔港背景
**核心记忆点：** 40年出海经验、认识每一艘渔船、凌晨3点码头选货

**内容方向：**
- 凌晨渔港实拍：跟船出海、码头拍卖、选货过程
- 渔民故事：老渔民的生活、渔船文化、海洋知识
- 源头揭秘：为什么超市贵、中间商赚差价、如何辨别新鲜度
- 真诚人设：不说假话、不好吃不卖、宁可亏本也要品质

**差异化卖点：**
- "我卖的每一批货，都是我凌晨3点在码头亲手挑的"
- "40年渔民，什么海鲜没见过？不好的我第一个不吃"
- "没有中间商，从渔船到你家"

**适合产品：** 虾皮、小鱼干、鱿鱼丝等干货类
**变现路径：** 源头直供信任 → 高复购率 → 会员体系

---

## B方案：海鲜猎人买手（严选品质路线）

**人设标签：** 海鲜猎人小李 / 海味买手阿杰 / 挑剔的海味师
**年龄定位：** 30-40岁男性
**形象特征：** 专业干练、品鉴装备、实验室风格
**核心记忆点：** 走遍沿海产地、严苛选品标准、专业检测设备

**内容方向：**
- 产地溯源：福建东山、浙江舟山、山东威海实地探访
- 专业测评：盐分检测、水分检测、添加剂检测
- 对比评测：超市货 VS 我们的货、价格对比、品质对比
- 知识科普：如何挑选、如何保存、营养价值

**差异化卖点：**
- "我跑遍中国海岸线，只为找到最好的虾皮"
- "每一批货都要过我这关，不合格直接退"
- "用买手的专业，帮你省掉试错成本"

**适合产品：** 中高端海鲜零食、礼盒装、进口海鲜
**变现路径：** 专业信任 → 高客单价 → 企业团购

---

## C方案：返乡创业青年（生活方式路线）

**人设标签：** 返乡阿明 / 渔村小森 / 海味青年
**年龄定位：** 25-35岁男性/女性均可
**形象特征：** 阳光活力、文艺清新、海边生活
**核心记忆点：** 大城市返乡、理想主义、慢生活倡导者

**内容方向：**
- 返乡故事：为什么放弃城市、渔村生活日常、创业心路
- 生活方式：海边日出、渔港慢生活、海鲜美食制作
- 产品场景：办公室零食、追剧伴侣、送礼佳品
- 情感共鸣：年轻人的焦虑、回归自然、简单生活

**差异化卖点：**
- "我在海边，把日子过成诗"
- "大城市卷不动了，回来卖海鲜也挺好"
- "不只是零食，是一种生活态度"

**适合产品：** 年轻化包装、小份装、联名款、文创周边
**变现路径：** 情感共鸣 → 年轻化客群 → 品牌IP化
