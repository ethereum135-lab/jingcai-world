## 二、首单推荐方案

### 推荐货源：FMG超特粉 10万吨

| 参数 | 建议值 |
|------|--------|
| **供应商** | Fortescue Metals Group (FMG) |
| **品种** | Super Special Fines（超特粉） |
| **数量** | 100,000 公吨（±10% 溢短装） |
| **品位** | Fe 58-61%，典型值59.5% |
| **价格基准** | 普氏62%铁矿石指数 |
| **定价公式** | 普氏62%指数 - 折扣（预估5-8美元/吨） |
| **参考价格区间** | 95-105美元/吨（FOB，基于当前市场预估） |
| **贸易条款** | FOB 黑德兰港，澳洲 |
| **目的港** | 青岛/日照/连云港（中国主港） |
| **付款方式** | 不可撤销即期信用证（L/C at sight） |
| **交货期** | 合同签订后30-45天装船 |

### 选择FMG的理由

| 优势 | 说明 |
|------|------|
| **准入门槛低** | 相比力拓/必和必拓，FMG对新客户更开放 |
| **价格优势** | 比62%主流粉矿低5-8美元/吨 |
| **供应稳定** | 年产量近2亿吨，履约能力强 |
| **条款灵活** | 付款和交货条件可谈判空间较大 |
| **适合首单** | 10万吨量级匹配FMG现货销售策略 |

### 风险提示

| 风险 | 等级 | 应对措施 |
|------|------|----------|
| 品位较低（58-61%） | 中 | 适合配矿使用，需确认下游接受度 |
| 折扣波动 | 中 | 合同中锁定折扣区间 |
| 汇率波动 | 中 | 信用证以美元计价，锁定汇率风险 |

## 三、对接路径

### 推荐路径（按优先级）

**路径1：通过大型贸易商（推荐）**
- 嘉吉（Cargill）、托克（Trafigura）、嘉能可（Glencore）
- 优势：渠道成熟、信用背书、流程标准化
- 劣势：需支付贸易溢价（约0.5-1美元/吨）

**路径2：直接与FMG销售部门对接**
- FMG新加坡销售公司负责亚洲业务
- 优势：价格更优、直接建立供应关系
- 劣势：需资质审核、首单门槛较高

**路径3：通过国内大型钢企/贸易商合作**
- 与有FMG长协的国内钢厂合作分货
- 优势：门槛低、周转快
- 劣势：利润空间受限
