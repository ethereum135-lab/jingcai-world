## 一、IP人设方案（3个方向）

### 方向A：「知识型赶海达人」⭐推荐
**人设定位：** 专业海洋生物知识科普 + 沉浸式赶海体验

**核心标签：**
- 海洋生物学爱好者/相关专业背景
- 10年+赶海经验的老渔民徒弟
- "海边的百科全书"

**差异化记忆点：**
1. **"海货鉴定师"** —— 每抓到一样东西都能说出学名、习性、最佳烹饪方式
2. **"潮汐预言家"** —— 根据月相预测最佳赶海时间，准确率高
3. **"海洋冷知识库"** —— 每期一个"你不知道的海洋秘密"

**人设台词：**
- "这不是普通的螃蟹，这是中华虎头蟹，只有在农历初三和十八的退潮时才会出来..."
- "老渔民说，看云识天气，看月相识潮汐，今天这个云，两小时后必有大潮..."

**内容风格：** 专业中带着烟火气，知识密度高但不枯燥，有教育意义又有解压感

---

### 方向B：「治愈系海边的慢生活家」
**人设定位：** 逃离城市喧嚣，在海边寻找生活本真的慢生活博主

**核心标签：**
- 前都市白领/设计师，裸辞来到海边
- "把日子过成诗"的生活美学践行者
- 海边小院改造达人

**差异化记忆点：**
1. **"日出收藏家"** —— 每天记录不同模样的海上日出
2. **"海边的一百种玩法"** —— 从捡贝壳到海玻璃DIY，发现海边的小确幸
3. **"今日份大海盲盒"** —— 每天海边散步，看看大海今天"送"什么礼物

**人设台词：**
- "城市里的我焦虑到失眠，海边的我听着浪声就能睡着..."
- "今天大海送了我一颗完美的猫眼螺，还有这个颜色的海玻璃，做成项链一定很美..."

**内容风格：** 治愈系、ASMR感强、画面唯美、文案走心，适合深夜emo人群

---

### 方向C：「硬核赶海实战派」
**人设定位：** 技术流赶海，追求效率和产量的实战型博主

**核心标签：**
- 职业赶海人/海鲜供应商
- "工具狂魔"，各种专业赶海装备测评
- 当地赶海比赛冠军

**差异化记忆点：**
1. **"赶海ROI计算"** —— 每次赶海记录投入产出比，几小时赚多少钱
2. **"装备测评师"** —— 专业赶海工具测评，从蟹笼到探照灯
3. **"赶海地图"** —— 揭秘各个海域的"藏货点"，哪里有什么货门儿清

**人设台词：**
- "今天3小时，成本50块，收获价值300块的海货，净赚250..."
- "这个蟹笼我测试了10个品牌，这个性价比最高，下货率..."

**内容风格：** 硬核、干货满满、数据化、实用性强，男性观众占比高

## 二、IP视觉识别系统（VIS）

### 1. 核心形象设计

#### 基础形象
- **年龄感：** 30-40岁，成熟稳重但不失活力
- **外貌特征：** 健康的小麦肤色（海边日晒痕迹）、略带沧桑但眼神明亮
- **标志性装扮：** 
  - 必备：宽檐渔夫帽（可绣"捕海翁"logo）
  - 上装：工装风防水外套/复古海魂衫
  - 下装：工装短裤/防水裤
  - 鞋子：橡胶水靴（可卷起裤脚露出）
  - 配饰：防水手表、工具腰包

#### 标志性道具
1. **"翁之铲"** —— 特制的多功能赶海铲，手柄缠有标志性的蓝色防滑绳
2. **"海货鉴定手册"** —— 一本破旧的笔记本，记录各种海货知识
3. **"潮汐罗盘"** —— 复古风格的指南针，作为开场/转场的视觉符号

### 2. 色彩系统

#### 主色调
- **深海蓝** `#0B3D4C` —— 代表专业、深度、信任感
- **沙滩金** `#D4A574` —— 代表收获、温暖、质感

#### 辅助色
- **珊瑚橙** `#FF7F50` —— 用于强调、按钮、互动元素
- **浪花白** `#F5F5F5` —— 背景、留白、呼吸感
- **海藻绿** `#2E8B57` —— 自然、生态、可持续

#### 色彩应用比例
- 深海蓝 40%（主背景、文字）
- 沙滩金 30%（强调、收获时刻）
- 浪花白 20%（留白、过渡）
- 珊瑚橙/海藻绿 10%（点缀、CTA）

### 3. 标志性视觉元素

#### Logo设计概念
- **图形：** 简化的人形剪影 + 渔网/海浪元素
- **字体：** 手写风格的"捕海翁"三个字，有毛笔质感但不失现代感
- **Slogan：** "潮起潮落，我在海边等你"

#### 视频视觉规范
- **封面模板：** 统一采用16:9，左侧人物/海货特写，右侧大面积留白放标题
- **标题字体：** 粗体手写风格，白色描边+阴影，确保在任何背景下清晰
- **转场动画：** 海浪冲刷效果、潮汐退去效果
- **字幕样式：** 圆角矩形底+白色文字，底部居中或跟随人物

#### 拍摄风格
- **滤镜：** 偏暖色调，增加一点复古胶片感，对比度适中
- **运镜：** 大量第一人称视角（POV），增强沉浸感
- **特写：** 海货细节、手部动作、表情反应
- **空镜：** 日出日落、海浪、海鸥、渔船

## 四、爆款内容公式

### 公式1：悬念钩子型
**适用：** 新奇特海货、罕见发现

**标题模板：**
- "海边发现不明生物，老渔民都说没见过！"
- "退潮后在礁石缝发现这个，价值上千？"
- "凌晨5点赶海，没想到遇到它..."

**开头公式（前3秒）：**
```
画面：特写神秘海货/发现瞬间
台词："这是什么？！" / "你们绝对没见过！"
音效：惊讶音效 + 心跳声
```

**结构模板：**
1. **0-3秒：** 悬念钩子（最吸引人的画面/发现）
2. **3-10秒：** 背景介绍（时间、地点、情况）
3. **10-30秒：** 探索过程（紧张感、波折）
4. **30-45秒：** 揭晓答案（是什么、值多少钱、怎么处理）
5. **45-60秒：** 知识延伸/互动引导

**案例脚本：**
```
【画面】特写一只奇怪的蓝色生物
【台词】"这是什么鬼东西？！"
【字幕】"海边发现不明生物"

【画面】切换到第一人称视角，在礁石区寻找
【台词】"凌晨5点，我在老渔民都不来的这片礁石区，
        发现了这个..."

【画面】用铲子轻轻翻动，展示完整形态
【台词】"有人认识吗？评论区告诉我"

【画面】特写+对比图（正常版本）
【台词】"查了一下，这好像是变异的...
        正常应该是这个颜色"

【画面】放生或处理过程
【台词】"不管值不值钱，先放了吧，大海的馈赠要珍惜"

【画面】关注引导
【台词】"关注我，带你看更多海边的秘密"
```

### 公式2：知识干货型
**适用：** 教学、科普、攻略

**标题模板：**
- "老渔民不外传的赶海秘诀，今天全告诉你！"
- "学会这3招，赶海收获翻3倍"
- "为什么你赶海总是空手而归？问题出在这里"

**开头公式（前3秒）：**
```
画面：对比（新手vs老手收获）
台词："同样赶海3小时，为什么他收获这么多？"
字幕："秘诀来了"
```

**结构模板：**
1. **0-3秒：** 痛点/对比（引发共鸣）
2. **3-8秒：** 承诺（今天教你什么）
3. **8-35秒：** 干货内容（3-5个要点，每个配演示）
4. **35-50秒：** 实战验证（用刚学的技巧现场演示）
5. **50-60秒：** 总结+收藏引导

**案例脚本：**
```
【画面】左右分屏：左边满满一桶，右边空篮子
```
